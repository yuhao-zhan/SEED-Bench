# Memory Index

## When to Read Which Memory File

| File | When to Read | Why |
|------|-------------|-----|
| `memory/user_preferences.md` | When starting a new project session or modifying workflow | Project org rules, code style, harness vs benchmark distinction |
| `memory/shared_audit_rules.md` | Before running auto_audit.sh or auto_feedback.sh; when auditing variable classification | Defines Constraint > Visible > Invisible rules, UNIFORM_SUFFIX, _CE mode |

## Files
